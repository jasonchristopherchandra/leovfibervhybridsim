# leovfibervhybridsim
A SimPy based simulation that simulates network RTTs, packet loss, and jitter for testing different network architectures and different distance profiles when used for intercontinental gaming. **NumPy** & **Matplotlib** is used for data analysis and visualization.

## Requirements

- Python 3.8 or higher
- macOS / Linux / Windows
- Python packages:
  - `simpy`
  - `numpy`
  - `matplotlib`

---

## Installation

1. **Clone or download this repository**.
2. **Install required packages** via pip:

```bash
pip install simpy numpy matplotlib
